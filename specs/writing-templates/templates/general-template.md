# 通用公文模板

## 模板说明

本模板适用于无法归类到特定场景的正式文书。

## 文书结构

```
                    {{title}}

{{recipient}}：

    {{opening}}

    {{body}}

    {{closing}}

                                        {{signature}}
                                        {{date}}
```

## 各部分说明

### 标题
- 居中显示
- 概括文书主旨
- 格式：关于XXX的XXX

### 收文对象（如适用）
- 顶格书写
- 加冒号

### 开头
- 说明缘由或背景
- 引出正文内容

### 正文
- 分段陈述
- 层次分明
- 重点突出

### 结尾
- 总结或表态
- 提出期望或请求

### 落款
- 单位名称
- 日期

## 变量占位符

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{title}}` | 文书标题 | 关于XXX的说明 |
| `{{recipient}}` | 收文对象 | 各部门 |
| `{{opening}}` | 开头段落 | 为进一步... |
| `{{body}}` | 正文内容 | （具体内容） |
| `{{closing}}` | 结尾段落 | 特此说明 |
| `{{signature}}` | 落款单位 | XXX办公室 |
| `{{date}}` | 落款日期 | 2024年X月X日 |

## 写作要点

1. **开门见山**：直接说明目的
2. **条理清晰**：分点陈述
3. **语言规范**：使用书面语
4. **格式统一**：遵循公文格式

## 常用结语

- 特此通知
- 特此说明
- 请予以支持
- 请予以配合
- 此致 敬礼
