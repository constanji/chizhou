# 工作报告/总结模板

## 模板说明

适用于：年度总结、季度总结、月度总结、调研报告、述职报告、专项工作报告等。

## 文书结构

```
                关于{{topic}}的{{report_type}}

{{recipient}}：

    {{opening_paragraph}}

    一、{{section1_title}}

    {{section1_content}}

    二、{{section2_title}}

    {{section2_content}}

    三、{{section3_title}}

    {{section3_content}}

    四、下一步工作计划

    {{next_steps}}

                                        {{signature}}
                                        {{date}}
```

## 标准结构

### 一、工作概述
- 时间范围
- 工作背景
- 总体情况

### 二、主要工作及成效
- 重点工作一
  - 具体做法
  - 取得成效
  - 典型案例
- 重点工作二
  - ...
- 重点工作三
  - ...

### 三、存在问题及分析
- 问题一及原因
- 问题二及原因
- 客观分析

### 四、下一步工作计划
- 工作思路
- 具体措施
- 目标任务

## 变量占位符

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{topic}}` | 报告主题 | 2024年度工作 |
| `{{report_type}}` | 报告类型 | 总结/报告/汇报 |
| `{{recipient}}` | 报告对象 | 局领导 |
| `{{period}}` | 时间范围 | 2024年1-6月 |
| `{{signature}}` | 落款单位 | XXX处 |
| `{{date}}` | 落款日期 | 2024年X月X日 |

## 写作要点

### 开头段落
- 概述报告背景和目的
- 说明时间范围
- 总括主要情况

**示例**：
> 根据XXX工作部署，现将2024年度工作情况报告如下：

### 主体部分
- 突出重点工作
- 用数据说话
- 体现成效

**示例**：
> 一年来，在局党组的正确领导下，我处紧紧围绕中心工作，扎实推进各项任务，取得了明显成效。全年共完成XXX项，同比增长XX%...

### 问题分析
- 客观分析
- 不回避问题
- 找准原因

### 工作计划
- 思路清晰
- 措施具体
- 目标明确

## 语气风格

- **正式严谨**：向上级汇报
- **实事求是**：有成绩讲成绩，有问题讲问题
- **条理分明**：层次清楚，重点突出

## 常用表述

### 开头
- 根据XXX要求/部署/安排
- 按照XXX精神
- 为贯彻落实XXX

### 过渡
- 一是...二是...三是...
- 首先...其次...再次...
- （一）...（二）...（三）...

### 结尾
- 以上报告如有不妥，请批评指正
- 特此报告
- 以上汇报，请审阅
