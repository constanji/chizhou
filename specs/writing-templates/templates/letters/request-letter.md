# 请示/函件模板

## 模板说明

适用于：请示、批复、函、通知、通报、公函等。

## 请示模板

```
                关于{{topic}}的请示

{{recipient}}：

    {{background}}

    {{request_content}}

    {{closing}}

    附件：{{attachments}}

                                        {{signature}}
                                        {{date}}
```

## 函件模板

```
                关于{{topic}}的函

{{recipient}}：

    {{opening}}

    {{body}}

    {{closing}}

                                        {{signature}}
                                        {{date}}
```

## 通知模板

```
                关于{{topic}}的通知

{{recipient}}：

    {{body}}

    {{requirements}}

    {{closing}}

                                        {{signature}}
                                        {{date}}
```

## 标准结构

### 请示类
1. 标题：关于XXX的请示
2. 主送机关
3. 请示缘由
4. 请示事项
5. 结语
6. 落款

### 函件类
1. 标题：关于XXX的函
2. 主送机关
3. 来函背景/发函目的
4. 具体内容
5. 结语
6. 落款

### 通知类
1. 标题：关于XXX的通知
2. 主送机关
3. 通知内容
4. 工作要求
5. 结语
6. 落款

## 变量占位符

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{topic}}` | 事项主题 | 申请XX经费 |
| `{{recipient}}` | 主送机关 | 市财政局 |
| `{{background}}` | 背景说明 | 根据XX需要... |
| `{{request_content}}` | 请示事项 | 拟申请XX万元 |
| `{{closing}}` | 结语 | 妥否，请批示 |
| `{{signature}}` | 落款单位 | XXX局 |
| `{{date}}` | 日期 | 2024年X月X日 |

## 写作要点

### 请示

**缘由部分**：
- 说明背景情况
- 阐述请示理由
- 引出请示事项

**示例**：
> 为贯彻落实XXX精神，进一步加强XXX工作，根据《XXX》要求，我局拟开展XXX活动。现将有关情况报告如下：

**事项部分**：
- 明确请示内容
- 说明具体要求
- 附必要依据

**示例**：
> 经研究，拟申请专项经费XX万元，用于XXX工作。其中：XXX费用XX万元，XXX费用XX万元。

**结语**：
- 妥否，请批示
- 以上请示当否，请批复
- 特此请示，请予批准

### 函件

**开头**：
- 说明发函目的
- 引用来函（复函）

**示例**：
> 贵单位《关于XXX的函》（X函〔2024〕X号）收悉。经研究，现答复如下：

**正文**：
- 陈述具体事项
- 表明态度意见
- 提出建议请求

**结语**：
- 请予以支持/协助
- 特此函复/函告
- 请研究办理见复

### 通知

**正文**：
- 通知事项明确
- 要求具体清晰
- 时限规定明确

**示例**：
> 为做好XXX工作，现将有关事项通知如下：
> 
> 一、XXX
> 二、XXX
> 三、XXX
> 
> 请各单位高度重视，认真贯彻落实。

**结语**：
- 特此通知
- 请遵照执行
- 请认真贯彻落实

## 语气风格

| 文种 | 语气特点 |
|------|----------|
| 请示 | 谦恭请求 |
| 批复 | 明确肯定 |
| 函 | 平等协商 |
| 通知 | 明确指令 |
| 通报 | 表彰/批评 |

## 格式规范

1. 标题格式：关于XXX的XXX
2. 主送机关顶格，加冒号
3. 正文首行缩进两格
4. 落款右下角，单位在上，日期在下

## 常用结语

### 请示类
- 妥否，请批示
- 以上请示当否，请批复
- 特此请示，请予批准
- 如无不妥，请批转各有关部门执行

### 函件类
- 请予以支持为盼
- 请予以协助配合
- 特此函复
- 请研究办理见复

### 通知类
- 特此通知
- 请遵照执行
- 请认真贯彻落实
- 请按此办理
