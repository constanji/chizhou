# 公文格式规范指南

## 概述

本指南提供公文写作的格式规范参考，帮助生成规范的公文内容。

## 标题格式

### 标题组成
公文标题一般由三部分组成：发文机关 + 事由 + 文种

**示例**：
- 关于开展XX工作的通知
- XX区人民政府关于XX的批复
- 关于申请XX经费的请示

### 标题要求
1. 准确概括公文内容
2. 简洁明了，一般不超过50字
3. 不用标点符号（书名号、引号除外）
4. 居中排列

## 正文格式

### 层次序号

| 层级 | 格式 | 示例 |
|------|------|------|
| 第一层 | 一、二、三、 | 一、工作目标 |
| 第二层 | （一）（二）（三） | （一）总体目标 |
| 第三层 | 1. 2. 3. | 1. 具体措施 |
| 第四层 | （1）（2）（3） | （1）第一步 |

### 段落格式
1. 首行缩进两个字符
2. 段落之间不空行
3. 层次标题与正文之间不空行

### 数字使用

| 情况 | 使用 | 示例 |
|------|------|------|
| 统计数据 | 阿拉伯数字 | 增长15.6% |
| 日期 | 阿拉伯数字 | 2024年3月15日 |
| 序号层次 | 汉字/阿拉伯 | 一、/1. |
| 概数 | 汉字 | 十几个、几百万 |
| 成语惯用语 | 汉字 | 三令五申、十分重要 |

## 常用文种格式

### 请示格式

```
标题（居中）

主送机关：（顶格）

    缘由段落...（首行缩进）

    请示事项段落...

    结语（妥否，请批示。）

                                落款单位
                                日期
```

### 报告格式

```
标题（居中）

主送机关：（如有）

    开头段落...

    一、第一部分标题
    
    内容...

    二、第二部分标题

    内容...

                                落款单位
                                日期
```

### 通知格式

```
标题（居中）

主送机关：

    正文内容...

    一、具体事项一
    
    二、具体事项二

    结语（特此通知。）

                                落款单位
                                日期
```

## 落款格式

### 单位落款
1. 位于正文右下方
2. 与正文空一行
3. 单位全称，不用简称

### 日期落款
1. 位于单位名称下方
2. 与单位名称对齐
3. 格式：XXXX年X月X日

**示例**：
```
                                        XX区人民政府办公室
                                        2024年3月15日
```

## 附件格式

### 附件说明
在正文之后、落款之前标注：

```
    附件：1. XXXXX
          2. XXXXX
```

### 附件要求
1. 附件名称与正文中表述一致
2. 多个附件用序号标注
3. 附件应单独成页

## 常用标点

| 标点 | 使用场景 | 示例 |
|------|----------|------|
| 顿号 | 并列词语 | 甲、乙、丙三方 |
| 逗号 | 句中停顿 | 根据工作需要，现通知如下 |
| 分号 | 分项叙述 | 一是...；二是...；三是... |
| 句号 | 句末 | 特此通知。 |
| 冒号 | 总起下文 | 具体要求如下： |

## 常见问题

### 1. 标题太长
- 精简内容，突出重点
- 删除修饰性词语
- 使用更精准的表达

### 2. 层次混乱
- 统一使用标准序号
- 同级内容同等缩进
- 避免跳级使用

### 3. 用词不规范
- 使用规范公文用语
- 避免口语化表达
- 数字使用规范

### 4. 落款不规范
- 使用单位全称
- 日期格式正确
- 位置对齐
