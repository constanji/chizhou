# 语气风格指南

## 概述

公文写作的语气风格需要根据文种类型、收文对象、写作目的等因素综合确定。

## 四种主要语气风格

### 1. 正式严谨（formal）

**适用场景**：
- 向上级领导汇报
- 重要决策文件
- 政策性文件
- 对外正式函件

**语言特点**：
- 用词规范，避免口语化
- 句式完整，不用省略句
- 措辞谨慎，表述准确
- 敬语使用得当

**示例表述**：
- ✓ 根据贵单位的要求，现将有关情况汇报如下
- ✗ 按照你们的要求，情况是这样的
- ✓ 经研究，拟采取以下措施
- ✗ 我们想了几个办法

**常用词汇**：
| 口语 | 正式表达 |
|------|----------|
| 想 | 拟、考虑 |
| 要 | 须、应当 |
| 给 | 予以、给予 |
| 很多 | 诸多、众多 |
| 马上 | 立即、迅速 |

### 2. 平实稳重（semi-formal）

**适用场景**：
- 一般工作文书
- 内部通知
- 工作简报
- 同级部门之间函件

**语言特点**：
- 平实自然
- 简洁明了
- 条理清晰
- 不卑不亢

**示例表述**：
- 根据工作安排，现将有关事项通知如下
- 经研究，同意你单位的请求
- 请各部门按要求做好相关工作

### 3. 专业务实（professional）

**适用场景**：
- 技术报告
- 调研分析
- 专项工作汇报
- 业务文书

**语言特点**：
- 术语使用准确
- 数据支撑充分
- 分析客观理性
- 建议务实可行

**示例表述**：
- 通过对XX数据的分析，可以得出以下结论
- 从技术层面看，该方案具有可行性
- 建议采用XX方式，预计可提升效率XX%

### 4. 热情亲切（warm）

**适用场景**：
- 欢迎词
- 致辞
- 感谢信
- 表彰通报

**语言特点**：
- 热情洋溢
- 真诚友好
- 感情充沛
- 鼓舞人心

**示例表述**：
- 值此新春佳节之际，向大家致以诚挚的问候和美好的祝愿！
- 在此，我代表XX，向长期以来关心支持我们工作的各位领导和朋友们表示衷心的感谢！

## 语气选择原则

### 根据收文对象选择

| 收文对象 | 建议语气 | 特点 |
|----------|----------|------|
| 上级领导 | 正式严谨 | 恭敬、谦和 |
| 同级部门 | 平实稳重 | 平等、协商 |
| 下级单位 | 平实/正式 | 指导、明确 |
| 外部单位 | 正式/专业 | 礼貌、专业 |

### 根据文种选择

| 文种 | 建议语气 |
|------|----------|
| 请示 | 正式严谨 |
| 报告 | 正式/专业 |
| 通知 | 平实稳重 |
| 讲话 | 热情/正式 |
| 函件 | 平实/正式 |
| 简报 | 平实/专业 |

## 注意事项

1. **避免语气不一致**：全文语气应保持统一
2. **避免过于生硬**：正式不等于冷漠
3. **避免过于随意**：公文应保持一定的正式度
4. **注意场合适配**：根据具体场合调整

## 敬语使用

### 对上级
- 呈、报、请示、汇报
- 贵（单位）、尊（敬语）
- 恳请、敬请

### 对下级
- 通知、要求、部署
- 各（单位）
- 望、请

### 平级
- 商洽、函告、函复
- 贵（单位）
- 请予以、盼复
