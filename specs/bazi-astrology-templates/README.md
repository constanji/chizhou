# 生辰八字/星座查询结果模板系统

## 概述

本模板系统为生辰八字（八字命理）和星座查询结果提供一套完整的文档模板、命令模板和检查清单模板。系统基于MCP工具中的知识，确保生成的查询结果规范、一致且易于使用。

## 目录结构

```
specs/bazi-astrology-templates/
├── templates/
│   ├── bazi-chart-template.md           # 八字命盘结果模板
│   ├── bazi-analysis-template.md        # 八字分析结果模板
│   ├── bazi-forecast-template.md        # 八字运势预测模板
│   ├── astrology-report-template.md     # 星座分析报告模板
│   ├── combined-analysis-template.md    # 综合命理分析模板
│   └── commands/
│       ├── bazi-generate-command.md     # 八字生成命令模板
│       ├── bazi-analyze-command.md      # 八字分析命令模板
│       └── astrology-command.md         # 星座分析命令模板
├── scripts/
│   ├── bash/

```

## 模板说明

### 1. 八字命盘结果模板 (bazi-chart-template.md)
用于生成标准的八字命盘结果，包含：
- 八字四柱（年柱、月柱、日柱、时柱）
- 十神配置
- 大运排盘
- 命宫、身宫等基本信息

### 2. 八字分析结果模板 (bazi-analysis-template.md)
用于生成八字命理分析报告，包含：
- 性格特征分析
- 事业财运分析
- 感情婚姻分析
- 健康运势分析

### 3. 八字运势预测模板 (bazi-forecast-template.md)
用于生成未来运势预测报告，包含：
- 大运流年分析
- 重要年份提醒
- 运势趋势预测
- 建议与注意事项

### 4. 星座分析报告模板 (astrology-report-template.md)
用于生成星座分析报告，包含：
- 星座基本信息
- 性格特点分析
- 运势周期分析
- 兼容性分析

### 5. 综合命理分析模板 (combined-analysis-template.md)
用于生成八字与星座的综合分析报告，包含：
- 八字与星座对比分析
- 命理一致性评估
- 综合运势预测
- 个性化建议

### 6. 命令模板 (commands/)
提供标准化的命令执行模板，确保不同工具生成的查询结果格式一致。

## 占位符系统

### 动态占位符（运行时替换）
- `$BIRTH_TIME`：出生时间
- `$GENDER`：性别
- `$QUESTION`：用户问题
- `$ANALYSIS_TYPE`：分析类型
- `$TIME_FRAME`：时间范围

### 静态占位符（模板填充时替换）
- `[用户姓名]`：用户姓名
- `[出生日期]`：出生日期
- `[出生地点]`：出生地点
- `[分析日期]`：分析日期
- `[星座名称]`：星座名称
- `[八字四柱]`：八字四柱信息

### 命理专用占位符
- `[年柱]`、`[月柱]`、`[日柱]`、`[时柱]`：八字四柱
- `[十神配置]`：十神关系
- `[大运排盘]`：大运信息
- `[命宫]`、`[身宫]`：命宫身宫
- `[五行强弱]`：五行平衡分析
- `[喜用神]`：喜用神分析

## 使用说明

### 1. 选择模板
根据查询需求选择合适的模板：
- 八字命盘查询 → bazi-chart-template.md
- 八字命理分析 → bazi-analysis-template.md
- 运势预测 → bazi-forecast-template.md
- 星座分析 → astrology-report-template.md
- 综合命理分析 → combined-analysis-template.md

### 2. 填充占位符
使用实际数据替换模板中的占位符：
```markdown
**出生时间**：`[出生日期]` → **出生时间**：`2004年3月22日`
**性别**：`[性别]` → **性别**：`女`
```

### 3. 生成报告
按照模板结构填充内容，确保所有必需部分都已填写。

### 4. 质量检查
使用检查清单模板验证生成报告的质量：
- 数据准确性检查
- 格式一致性检查
- 内容完整性检查
- 建议合理性检查
