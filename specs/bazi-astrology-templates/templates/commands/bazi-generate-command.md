## 概述

目标：根据用户提供的出生信息生成标准的八字命盘

注意：此命令需要准确的出生时间、性别和出生地点信息。如果信息不完整，需要向用户询问补充信息。

执行步骤：

1. **验证输入信息**
   - 检查是否提供了出生时间
   - 检查是否提供了性别信息
   - 检查出生时间格式是否正确
   - 如果信息不完整，向用户询问缺失信息

2. **处理出生信息**
   - 解析出生时间（支持阳历和农历）
   - 确定出生时辰（如果未提供具体时间）
   - 处理出生地点信息（用于精确计算）

3. **生成八字命盘**
   - 计算八字四柱（年柱、月柱、日柱、时柱）
   - 确定十神配置
   - 排定大运
   - 计算命宫、身宫
   - 分析五行强弱

4. **格式化输出**
   - 按照八字命盘模板格式化输出
   - 确保所有占位符都被正确替换
   - 保持输出格式的一致性

5. **验证结果**
   - 检查八字计算的准确性
   - 验证输出格式的正确性
   - 确保所有必需信息都已包含

6. **提供后续建议**
   - 建议进行八字分析
   - 建议进行运势预测
   - 提供个性化建议

行为规则：

- **数据准确性**：必须确保八字计算的准确性
- **格式一致性**：必须按照模板格式输出结果
- **信息完整性**：必须包含所有必需的命盘信息
- **用户友好**：输出应该易于理解和使用
- **错误处理**：如果输入信息有误，应该提供明确的错误信息和修正建议
