## 概述

变量使用示例文档通过具体的场景示例，展示如何在实际内容中使用变量替换。每个示例包括原始模板、变量替换过程、替换后结果，以及使用说明和注意事项。

## 基础使用示例

### 示例1：简单变量替换

**原始模板**：
```
今天[时间]去了[地点]，感觉[心情形容词]！
```

**变量说明**：
- `[时间]`：动态时间变量
- `[地点]`：静态地点变量  
- `[心情形容词]`：静态情感变量

**替换过程**：
1. 确定具体时间：下午3:30
2. 确定具体地点：北京故宫
3. 确定心情描述：非常开心

**替换后结果**：
```
今天下午3:30去了北京故宫，感觉非常开心！
```

**使用说明**：
- 时间变量可以使用具体时间或相对时间
- 地点变量要具体明确
- 心情形容词要符合实际感受

### 示例2：多变量组合

**原始模板**：
```
和[人物]一起在[地点][活动描述]，[时间]的[生活场景]真美好！
```

**变量说明**：
- `[人物]`：静态人物变量
- `[地点]`：静态地点变量
- `[活动描述]`：静态活动变量
- `[时间]`：动态时间变量
- `[生活场景]`：静态场景变量

**替换过程**：
1. 确定人物：张三、李四
2. 确定地点：咖啡厅
3. 确定活动：喝咖啡聊天
4. 确定时间：周末下午
5. 确定场景：悠闲时光

**替换后结果**：
```
和张三、李四一起在咖啡厅喝咖啡聊天，周末下午的悠闲时光真美好！
```

**使用说明**：
- 多个变量要逻辑连贯
- 活动描述要具体生动
- 场景描述要营造氛围
